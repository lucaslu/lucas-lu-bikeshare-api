API Documentation

Bikes API End Points

  • GET all Bikes - /bikes
  • GET one Bike Details - /bikes/:bikeId
  • POST a new Bike - /bikes. Post Object Structure 
    {
  name: bike-name,
  description: bike-description,
  address: address,
  city: city,
  host_name: name-host,
  price: price-bike
}
